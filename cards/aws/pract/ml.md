### Front
What is Amazon Lex and what is its primary use case?

### Back
Amazon Lex is a service for building conversational interfaces (chatbots) using voice and text. It uses the same deep learning technologies that power Amazon Alexa. Primary features:
- Natural Language Understanding (NLU)
- Automatic Speech Recognition (ASR)
- Integration with AWS Lambda for business logic
- Scalable architecture for chat applications

<!-- Card End -->

<!-- Card Start -->
### Front
Explain the key components of an Amazon Lex bot configuration.

### Back
An Amazon Lex bot configuration consists of:
- **Intents**: What the user wants to do
- **Utterances**: Different ways users might express their intent
- **Slots**: Required information to fulfill the intent
- **Fulfillment**: Actions to take when intent is complete
- **Response**: How the bot responds to the user
